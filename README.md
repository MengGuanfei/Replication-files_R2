# Replication-files_R2
This replication package (Green or Growth? Unveiling the Eco-environmental Consequence of Cross-border E-commerce Pilots in China) provides all the publicly available data, code, andinstructions required to reproduce the main empirical results of the above-mentionedresearch paper. The package is designed to facilitate transparency and enableindependent verification of all statistical findings reported in the study.
